# Tareas

## ¿Qué es?
Gestión de tareas e hitos del equipo legal. Permite revisar las tareas propias y las del equipo organizadas por estado. Las tareas se crean desde dentro de cada expediente.

## ¿Quién lo usa?
- Abogados: revisan el progreso de las tareas asignadas a su equipo
- Practicantes: revisan las tareas que tienen pendientes

## Cómo acceder
Menú lateral → Tareas

---

## Pantalla principal

Ruta: Tablero Principal > Tareas

- Título: "Tareas"
- Subtítulo: "Revisa tus tareas y las de tu equipo."

**Botón principal:**

- Botón "Mis Hitos" (barra oscura ancha): filtra para mostrar solo los hitos asignados al usuario logueado

**Filtros de estado:**

| Filtro | Descripción |
| --- | --- |
| Pendiente + En Curso | Muestra tareas pendientes y en curso (seleccionado por defecto) |
| Pendiente | Muestra solo tareas pendientes |
| En Curso | Muestra solo tareas en progreso |
| Completado | Muestra solo tareas completadas |

**Filtro de expediente:**

- Selector "Todos los expedientes" para filtrar tareas por expediente específico

**Ordenamiento:**

- Selector "Más atrasados primero" para ordenar las tareas por fecha límite

**Estado vacío:**

Muestra mensaje "No tienes hitos con los filtros seleccionados." cuando no hay tareas que coincidan con los filtros aplicados.

---

## Nota importante
Las tareas no se crean desde este módulo. Se crean desde dentro de cada expediente y luego aparecen aquí para ser revisadas.
